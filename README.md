# Instalacao

## Em ambiente codespaces

```
pipx install poetry
```

```
poetry shell
```

```
poetry run python3 app.py
```

## Em ambiente local

```
curl -sSL https://install.python-poetry.org | python3 -
```

```
~/.local/share/pypoetry/venv/bin/poetry shell
```

```
~/.local/share/pypoetry/venv/bin/poetry run python3 app.py
```
